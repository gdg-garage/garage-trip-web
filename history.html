About the Event

This trip is an initiative of GDG Garage Prague One attendees.

We felt that more time and focus were needed for some algorithmic tasks that we tried to solve at weekly GDG Garage meetings.

Garage Trip is non-profit event powered by sheer joy of the organizers and attendees.

The aim of the trip is to have fun and relax, which is supported by:

  • Solving algorithmic tasks (in pairs and teams)
  • Outdoor games
  • LAN party
  • Board games
  • Logic (ciphering) games - puzzlehunt, escape game
